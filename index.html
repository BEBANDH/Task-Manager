      📜 Changelog & App Features

      v5.0 (Current Version) July 2026
      • Unified Dark-Only UI: Completely removed Light Mode, focusing all visuals on a premium Graphite Dark base with toggleable AMOLED Pure Black overrides.
      • Overhauled Progress Timer: Upgraded the right sidebar timer to a circular progress card. Added inline quick-adjusters (-10m, -5m, +5m, +10m) for rapid duration setting.
      • Accent-Bound Progress Ring: Connected the countdown circle's stroke and glow shadow directly to the user's active Accent color selector.
      • Header & Sidebar Migrations: Migrated the Dark Mode switch, Changelog trigger, Accent pickers, and keyboard shortcuts sheets from the main header and sidebar into the dedicated Settings panel to clean up the workspace.
      • Large Day-Labeled Heatmap: Increased the Dashboard's horizontal contribution chart sizing, aligning Sun-Sat row offsets clearly on the Y-axis.
      • Compact Rectangular Cards: Redesigned the task board list to a 2-column compact grid, shifting task titles to full line spaces and metadata below them.
      • Solid Selection Controls: Re-engineered all checkboxes and accent color indicators to use solid fills and borders without tick-mark characters to maximize clean lines.
      v4.0 July 2026
      • Dashboard Bulk Actions: Moved "Clear Completed" to the Dashboard and added selective multi-list clearing.
      • Multi-Sheet Excel Import: You can now import an Excel file containing multiple sheets; each sheet is created as a separate list.
      • Premium Design & Theme Overhaul: Upgraded light mode to use soft alpha-transparent borders, and dark mode to a slate-900 off-black background. Added dynamic accent gradients for progress bars, buttons, and glowing active SVG heatmap days.
      • Elevated Task Card Mechanics: Task cards now float with subtle shadows and pop upwards on hover. Checkboxes feature a custom CSS drawing checkmark on toggle, and completed text dims without strikethroughs to preserve readability.
      • Snappy Search & Disk Debouncing: Debounced typing in all search inputs (250ms/200ms) and disk writes (300ms via requestIdleCallback) to remove stutters.
      • Firebase API Write Batching: Postpones Firestore cloud syncs by 1.5 seconds of user inactivity, reducing redundant Firebase API usage by up to 90% during quick edits.
      • Incremental DOM Updates: Completing tasks modifies DOM elements directly instead of completely rebuilding the lists, rendering checkbox toggles instantly.
      • List Cycling Shortcut: Added Alt+T to quickly cycle through your task lists.
      • UI Fixes: Improved styling of modal checkboxes and expanded touch target boundaries for actions.
      v3.0 July 2026
      • Premium Visual Overhaul: Redesigned task lists with card-based layouts, soft hover elevations, and a sleek spotlight "Add Task" input.
      • GitHub-Style Heatmap: Replaced the legacy bar chart with a responsive 365-day contribution heatmap graph and smart tooltips.
      • Sleek Iconography: Removed all text emojis and replaced them with crisp, cohesive SVG icons across the entire app.
      • Fluid Micro-Interactions: Added tactile checkbox animations, smooth strikethroughs, and fluid transitions.
      • UI Polish: Whisper-thin adaptive scrollbars, custom CSS tooltips, and beautifully designed empty states.
      v2.0 July 2026
      • Inline Dashboard & Settings: Removed popup modal, replacing it with an integrated full tab view for clean layout.
      • Modifications Locking: Lock toggle button (L key) to freeze specific lists to prevent edit, delete, complete or priority changes.
      • Activity Chart Visibility: Quick toggle button directly on page header to hide or show completed tasks chart.
      • Extended Analytics: Added metrics tracking for Max Daily Streak, Active Tasks Remaining, and Average Task Completion Time.
      • Sidebar Toggle Shortcut: Dedicated shortcut key (S) to collapse/expand left panels.
      • Accent Color Cycling: Press C to cycle through all theme accent colors.
      v1.5 May 2026
      • Activity Progress Chart: Added SVG line chart showing task completions throughout the current year.
      • Excel Import/Export: Enabled bulk backups and migrations to/from spreadsheet documents.
      • Subtasks Support: Ability to split lists down into sub-checklists inside each task item.
      v1.0 (Initial Release) Early 2026
      • Multiple Lists: Dynamic sidebar for organizing tasks by groups (Work, Personal, etc.).
      • Theme & Accents: Minimalist theme selectors and custom accent highlight colors.
      • Keyboard Shortcuts: Work-efficiency hotkeys (N for new task, / for search, T for theme).
