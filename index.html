      📜 Changelog & App Features

      v4.0 (Current Version) July 2026
      • Dashboard Bulk Actions: Moved "Clear Completed" to the Dashboard and added selective multi-list clearing.
      • Multi-Sheet Excel Import: You can now import an Excel file containing multiple sheets; each sheet is created as a separate list.
      • Saturated Themes & Layout: Light mode now utilizes the selected accent color as a heavily saturated pastel background. A new right-sidebar was added to create a clean, 3-column layout that spans the entire screen without any empty gaps.
      • Vertical 365-Day Heatmap: The contribution chart has been upgraded to a vertical grid (14 days per row), allowing it to fit beautifully in the right sidebar without scrolling. Newer tasks appear at the top, and empty days use a crisp, visible grid.
      • List Cycling Shortcut: Added Alt+T to quickly cycle through your task lists.
      • UI Fixes: Improved styling and layout of modal checklist items. Increased size of task action buttons for better clickability.
      v3.0 July 2026
      • Premium Visual Overhaul: Redesigned task lists with card-based layouts, soft hover elevations, and a sleek spotlight "Add Task" input.
      • GitHub-Style Heatmap: Replaced the legacy bar chart with a responsive 365-day contribution heatmap graph and smart tooltips.
      • Sleek Iconography: Removed all text emojis and replaced them with crisp, cohesive SVG icons across the entire app.
      • Fluid Micro-Interactions: Added tactile checkbox animations, smooth strikethroughs, and fluid transitions.
      • UI Polish: Whisper-thin adaptive scrollbars, custom CSS tooltips, and beautifully designed empty states.
      v2.0 July 2026
      • Inline Dashboard & Settings: Removed popup modal, replacing it with an integrated full tab view for clean layout.
      • Modifications Locking: Lock toggle button (L key) to freeze specific lists to prevent edit, delete, complete or priority changes.
      • Activity Chart Visibility: Quick toggle button directly on page header to hide or show completed tasks chart.
      • Extended Analytics: Added metrics tracking for Max Daily Streak, Active Tasks Remaining, and Average Task Completion Time.
      • Sidebar Toggle Shortcut: Dedicated shortcut key (S) to collapse/expand left panels.
      • Accent Color Cycling: Press C to cycle through all theme accent colors.
      v1.5 May 2026
      • Activity Progress Chart: Added SVG line chart showing task completions throughout the current year.
      • Excel Import/Export: Enabled bulk backups and migrations to/from spreadsheet documents.
      • Subtasks Support: Ability to split lists down into sub-checklists inside each task item.
      v1.0 (Initial Release) Early 2026
      • Multiple Lists: Dynamic sidebar for organizing tasks by groups (Work, Personal, etc.).
      • Theme & Accents: Minimalist theme selectors and custom accent highlight colors.
      • Keyboard Shortcuts: Work-efficiency hotkeys (N for new task, / for search, T for theme).
